## Quick Start

### Prerequisites

- Node.js 18+
- Docker & Docker Compose
- PostgreSQL database
- Redis

### Installation

1. Clone the repository
2. Copy `.env.example` to `.env` and configure
3. Run with Docker Compose:
   ```bash
   docker-compose up -d
   ```

### Development

```bash
# Backend
cd backend && npm run dev

# Frontend
cd frontend && npm run dev
```
